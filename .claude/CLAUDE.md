## Claude Code 特定指令

### 工具权限
- 读写 `wiki/` 目录下的 markdown 文件
- 只读 `raw/` 目录下的文件：可以读取，不得修改
- 创建新文件时遵循根 `CLAUDE.md` 中定义的页面约定

### 工作流执行
- **Ingest 前**：先与用户讨论要点，确认后再执行写入
- **更新多个页面时**：一次性完成所有关联页面的更新（源摘要、概念页、索引、日志），保持一致性
- **删除内容前**：先与用户确认

### 规则文件
`.claude/rules/` 下的路径范围规则按需加载。需要时可以创建新的规则文件，但需遵守根 `CLAUDE.md` 中的命名约定和 frontmatter 格式。
